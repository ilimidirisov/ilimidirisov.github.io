Git workflow

Git workflow может сделать так чтобы много людей работало именно с этого репозитория и не изменяет базовый репозиторий

Git rebase

Git rebase позволяет обьединить измененные две ветки в одну

Git fetch & Git pull

Git fetch загружает все file commits сайты в локальный репозиторий, но не меняет состояние нашего репозитория ,а git pull изменеят состояние нашего репозитория чтобы соответствовать ему

Git merge

Git merge обьединяет ветки
