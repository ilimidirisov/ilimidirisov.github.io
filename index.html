Что такое git workflow?

git workflow может сделать так чтобы много людей работало именно с этого репозитория и не изменяет базовый репозиторий

что такое git rebase?

git rebase позволяет обьединить измененные две ветки в одну

что твкое git fetch и git pull ?

git fetch загружает все файлы коммиты сайты в локальный репозиторий, но не меняет состояние нашего репозитория ,а git pull изменеят состояние нашего репозитория чтобы соответствовать ему

что такое git merge?

git merge просто обьединяет две ветки
